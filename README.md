# daily_reminder
给女朋友做的微信天气推送







模板内容如下：

{{date.DATA}} 

地区：{{region.DATA}} 

天气：{{weather.DATA}} 

气温：{{temp.DATA}} 

风向：{{wind_dir.DATA}} 

今天是我们恋爱的第{{love_day.DATA}}天 

{{birthday1.DATA}} 
{{birthday2.DATA}}


{{note_en.DATA}} 
{{note_ch.DATA}}



可以去天行数据申请各种各样的接口用来推送
